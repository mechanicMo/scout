How do you enter the mood search?

The chat bar currently sits at the bottom of the Picks screen. How should tapping it transition to the full mood search experience?

A

Bottom Sheet Expand

Tapping the chat bar expands a bottom sheet upward to full screen. The input stays at the top of the sheet. Feels like the keyboard pulled up a new layer. Swipe down to dismiss back to Picks.

Pros

  • Smooth, native-feeling transition
  • Easy to dismiss (swipe down)
  • Chat bar position stays consistent

Cons

  • Bottom sheet can feel cramped on smaller phones
  • Gesture conflicts with scrolling results
B

Full Screen Push

Tapping the chat bar navigates to a new full screen (standard stack push). Back button or swipe-back to return to Picks. Input bar at the top, history below.

Pros

  • Maximum space for content
  • Clean separation between Picks and Mood
  • Standard navigation pattern — no gesture conflicts

Cons

  • Slightly heavier transition
  • Losing sight of Picks feels like leaving the tab
C

Inline Transform

The Picks feed fades/slides away and the mood search UI takes its place in the same screen. The header changes from "Picks" to show the search context. No navigation push — it's a mode switch within the tab.

Pros

  • Stays within the Picks tab conceptually
  • Lightweight feel
  • No navigation stack complexity

Cons

  • Harder to implement cleanly
  • State management between modes gets tricky
  • "Back" behavior is less obvious
